# Fraud-detection-models
Machine learning project for fraud detection with ensemble models, MLflow tracking, PostgreSQL, and Grafana.
Данный проект представляет собой комплексную систему для обнаружения мошеннических транзакций в реальном времени. Система использует ансамбль моделей машинного обучения (CatBoost, XGBoost, RandomForest, LightGBM, MLP) для анализа транзакций и выявления подозрительных операций. Для отслеживания качества модели используется MLflow, данные хранятся в PostgreSQL, а для визуализации метрик настроена Grafana. Telegram-бот позволяет проверять отдельные транзакции и получать мгновенную оценку риска.

# Цель работы
Разработка и внедрение системы автоматического обнаружения мошеннических транзакций с использованием методов машинного обучения и современных инструментов MLOps.

# Описание проекта
Проект по обнаружению мошеннических транзакций с использованием ансамбля моделей машинного обучения. Система включает:

- MLflow для отслеживания экспериментов
